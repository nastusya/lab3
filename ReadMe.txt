1. Відкривши консолний рядок вводимо docker start new_sql_server 
2. В PyCharm  відкриваємо файл run.py та запускаємо його
3. Зайшовши по силці http://127.0.0.1:5000/users ви побачете список юзерів
4. http://127.0.0.1:5000/add-user ви можете додати потрібного вам користувача
5. Також вибравши потрыбного вам користувача,  ви можете выдредагувати його дані натиснувши на кнопочку edit
